Для создания новой базы данных в командной строке нужно ввести CREATE DATABASE название;
Прежде чем будет создавать таблицу, нужно выбрать базу данных. Это делается с помощью оператора USE название бд:
Для создания новой таблицы в командной строке нужно ввести CREATE TABLE название таблицы  
далее через запятые,в скобках список имен и определений каждого столбца таблицы: PRIMARY KEY= (имя_столбца) ) ENGINE= тип_движка;
